Apex_APIVersionUpdate
=====================

ApexCodeのAPIバージョン一括更新機能です。  
- APIバージョンの選択リスト値はカスタム設定で登録されます。
- 名前プレフィックスが存在するコードは検索対象外です。
- 名前検索は前方一致検索になります。
- 更新対象はページとコンポーネントです。クラスとトリガーは更新できません。
